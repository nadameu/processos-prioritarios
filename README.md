Summa Dies
==

Desenvolvimento
--

1. Após clonar o repositório:

```
npm install
```

2. Instalar a extensão Violentmonkey no Firefox.

3. Iniciar a compilação do script (abrirá automaticamente no navegador padrão):

```
npm start
```

Distribuição
--

```
npm run build
```

Os arquivos serão criados no diretório `dist`.
